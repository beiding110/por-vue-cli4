全局 `mixin` 文件

``` bash

mixins
    |-2way.js       双向数据流mixin
    |-dialog-form-page.js       使用了dialog的表单页面通用引用
    |-dialog-list-page.js       使用了dialog的列表页面通用引用
    |-form-page.js              表单页面通用引用
    |-list-page.js              列表页面通用引用
    |-model.js      双向绑定mixin

```

各个包内mixin文件应存放在 `包` 内管理；
