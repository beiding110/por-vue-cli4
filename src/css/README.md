全局`css文件`

``` bash
css
|- theme        element主题
    |-purple    主题色为紫色的主题(请勿修改，可前往官网重新生成)
    |-orange    主题色为橙色的主题(请勿修改，可前往官网重新生成)
|-connon.scss               项目内全局样式
|-element-customize.scss    element自定义样式
|-iconfont.css              图标文件
|-var.scss                  scss变量文件
|-zh-common.css             清除样式表全局样式(请勿修改)

```

各个包内css文件应存放在 `包` 内管理；
